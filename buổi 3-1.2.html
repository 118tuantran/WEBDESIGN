In CSS, a rule consists of a selector and a declaration block.

CSS Syntax Breakdown

The selector targets the HTML element you wish to style.

Within the declaration block, you'll find one or more declarations, each separated by semicolons.

Declarations pair a CSS property with a value, linked by a colon.

Remember to separate multiple declarations with semicolons and enclose the declaration block in curly braces.
